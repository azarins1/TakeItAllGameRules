Take It All!

Official Game Rules

Game Overview

Take It All! is a 4-player game with players red, green, yellow, and blue. Each player will start in a different color in the board, and they will then be allowed to move around on the board for a limited period of time. While moving on the board, players will turn squares that they cross over into their own color. Beware of other players, however; If two players collide, they both become a solid wall that can't move (and is now a hazard for other players). The objective of the game is to turn the largest amount of squares into the same color as you, and the player with the most squares to their color at the end of the set time period is declared the winner.

However, the twist is that players are not allowed to directly control their characters when the game is happening live. Instead, they must create a program that instructs their character how to move in every game frame. Below we have all of the coding functions fully explained, with several examples.

Movement Functions

player.turn_cw()

Turns the player 90 degrees clockwise

player.turn_ccw()

Turns the player 90 degrees counterclockwise

player.turn_random_direction()

Makes player turn random direction

player.turn_180()

Turns the player 180 degrees

player.face_right()

player.face_left()

player.face_up()

player.face_down()

Makes player change to specified direction (left, right, up, or down)

Sensory Functions

player.facing_wall()

Boolean return Value; It returns true when the player is facing the wall (as in it will run into the wall if it goes foreward one block) and false when it is not

player.facing_empty()

Boolean return Value; Returns true when player is facing an empty square (a white square), false otherwise.

player.facing_own_territory()

Boolean return Value; Returns true or false depending on whether the player is facing territory of own color (true if facing own color)

player.facing_enemy_territory()

Boolean return Value: Returns true if the player is facing an enemyâ€™s territory (marked by a color other than its own)

player.get_direction()

Integer return Value: Returns an integer depending on which direction the player is facing

0: Up

1: Right

2: Down

3: Left

player.get_x()

Integer return Value: Returns an integer based on x-coordinate location

Note: (x=1, y=1) is the bottom left corner, and (x=10, y=10) is the top right corner

player.get_y()

Integer return Value: Returns an integer based on y-coordinate location

player.enemy_dist_in_front()

Returns integer based on how much distance is between player and enemy
